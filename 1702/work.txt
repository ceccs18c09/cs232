       flexible  specification  of the set of files to be copied.  It is famous for its delta-transfer algorithm, which reduces the amount[m
       of data sent over the network by sending only the differences between the source files and the existing files in  the  destination.[m
       Rsync is widely used for backups and mirroring and as an improved copy command for everyday use.[m
